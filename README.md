# Repositorio Taller Express/Prisma

El servidor se puede iniciar usando 

```bash
npm run start
```

Veras un link con el siguiente formato
```bash
http://localhost:3000
```
Para la realización de consultas puedes usar los siguientes urls, esto usando insomnia y el metodo GET:
```bash
http://localhost:3000/users/
http://localhost:3000/users/:id
http://localhost:3000/tasks/
http://localhost:3000/tasks/user/:userId
```

Para la creación de las mismas puedes usar las siguientes urls, cambiando el metodo de GET a POST:
```bash
http://localhost:3000/users/
http://localhost:3000/tasks/
```
